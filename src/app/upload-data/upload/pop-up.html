Guide

How to select the final exam score file:
This file is in .csv format, normally exported directly from the Gradescope website. If your final score file is not from Gradescope, please make sure the csv file contains the column with student PID and student final exam score.

Please make sure to name the student PID column to be: SID
Please make sure to name the student final exam score column to be: Total Score (with a space in between)

How to select the iClicker folder:
1. Please go to the folder called “iclicker” generated by the iClicker software
2. Select the “classes” folder
3. Choose the folder with corresponding course and term
